集合与数组/映射的常用操作：聚合、过滤、转换、排序。

## 亮点

- 聚合：`Sum/Avg/Max/Min/Count` 以及安全变体
- 过滤/条件辅助
- 转换为数组/Fields/JSON
- 排序工具

## 兼容性

- Go `>= 1.25.0`
- 模块路径：`github.com/goal-web/collection`
